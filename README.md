# MiSTer Bubble Universe

### Intro
A graphical demo core for the MiSTer FPGA platform written in Verilog.

### Installation
To install it, copy the .rbf file from the **[Releases](https://github.com/movievertigo/MiSTer-BubbleUniverse/releases)** page to your MiSTer.

### Controls
The core is interactive, and you can map the following controls:
* **Left,Right,Up,Down :** *Pan controls*
* **In,Out :** *Zoom controls*
* **Faster,Slower,Pause :** *Animation speed*
* **Trails :** *Persistence*
